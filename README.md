
# Expense Tracker

This project is an Expense Tracker application, a windows application built using React.js, go, Tanstack Router, Echarts, Jotai, shadcn UI, Wails, and Pocketbase. It allows users to track their income and expenses, visualize analytics for their financial events, and export data to various formats such as CSV, Excel, and TXT. The application provides a user-friendly interface for managing financial transactions efficiently.

## Features

- Income and Expense Tracking: Users can easily add and categorize their income and expenses, helping them to keep track of their finances accurately.

- Analytics: The application provides graphical representations of financial data using Echarts, enabling users to gain insights into their spending habits over time.

- Export Data: Users can export their financial data to CSV, Excel, or TXT formats, allowing for seamless integration with other financial tools and software.


## Tech Stack

- **React.js**
- **Go**
- **Tanstack Router**

- **Echarts**

- **Jotai**

- **Wails**

- **Pocketbase**


## Run Locally

Clone the project

```bash
  git clone https://github.com/sankalpa-adhikari-sa/expense-tracker.git
```

Go to the project directory

```bash
  cd expense-tracker
```


Start the server

```bash
  wails dev
```
Build application

```bash
  wails build -nsis
```
Navigate to project directory ->build ->bin to get the .exe file
